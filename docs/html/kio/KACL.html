KACL Class Reference

from PyKDE4.kio import *

Detailed Description

The KACL class encapsulates a POSIX Access Control List. It follows the little standard that couldn't, 1003.1e/1003.2c, which died in draft status. a POSIX ACL encapsulation

Author:
Till Adam <adam@kde.org>
