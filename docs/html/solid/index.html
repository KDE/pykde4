solid Module

Solid

Solid is a device integration framework. It provides a way of querying and interacting with hardware independently of the underlying operating system.

Currently, only the HAL backend is implemented, so Solid will only work on Linux, but more backends will follow.

See the documentation for the Solid namespace, and the tutorial on TechBase.
