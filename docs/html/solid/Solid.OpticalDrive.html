Class Solid.OpticalDrive
Inherits Solid.StorageDrive

This device interface is available on CD-R*,DVD*,Blu-Ray,HD-DVD drives.

A Cdrom is a storage that can handle optical discs.

  • OpticalDrive (backendObject)
    returns nothing
    ArgumentTypeDefault
    backendObjectQObject

    This type stores an OR combination of MediumType values. Creates a new Cdrom object. You generally won't need this. It's created when necessary using Device.as().

    backendObject - the device interface object provided by the backend
