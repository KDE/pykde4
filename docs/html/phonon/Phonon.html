Enumeration Documentation

ObjectDescriptionType

Audio output devices. This can be soundcards (with different drivers), soundservers or other virtual outputs like playback on a different computer on the network.

For Hardware devices the backend should use libkaudiodevicelist (\ref AudioDevice and AudioDeviceEnumerator) which will list removable devices even when they are unplugged and provide a unique identifier that can make backends use the same identifiers.
