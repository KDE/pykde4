Class KIO.SimpleJob
Inherits KIO.Job

Forward signal from subjob. job - the subjob speed - the speed in bytes/s

See also speed() A simple job (one url and one command). This is the base class for all jobs that are scheduled. Other jobs are high-level jobs (CopyJob, DeleteJob, FileCopyJob...) that manage subjobs but aren't scheduled directly.
