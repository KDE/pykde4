HTMLAnchorElement Class Reference

from PyKDE4.khtml import *

Inherits: DOM.HTMLElement → DOM.Element → DOM.Node
Namespace: DOM

Detailed Description

The anchor element. See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#edef-A"> A element definition in HTML 4.0.

The character encoding of the linked resource. See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#adef-charset"> charset attribute definition in HTML 4.0.

DOM.DOMString coords (   self )

Comma-separated list of lengths, defining an active region geometry. See also shape for the shape of the region. See the <a href="http://www.w3.org/TR/REC-html40/struct/objects.html#adef-coords"> coords attribute definition in HTML 4.0.
