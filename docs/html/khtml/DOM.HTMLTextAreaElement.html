HTMLTextAreaElement Class Reference

from PyKDE4.khtml import *

Inherits: DOM.HTMLElement → DOM.Element → DOM.Node
Namespace: DOM

Detailed Description

Multi-line text field. See the <a href="http://www.w3.org/TR/REC-html40/interact/forms.html#edef-TEXTAREA"> TEXTAREA element definition in HTML 4.0.
