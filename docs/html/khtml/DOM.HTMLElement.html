class HTMLElement
Module khtml
Namespace DOM
Class HTMLElement
Inherits DOM::Element
All HTML element interfaces derive from this class. Elements that only expose the HTML core attributes are represented by the base HTMLElement interface. These elements are as follows:

  • HEAD

  • special: SUB, SUP, SPAN, BDO

  • font: TT, I, B, U, S, STRIKE, BIG, SMALL

  • phrase: EM, STRONG, DFN, CODE, SAMP, KBD, VAR,
    • CITE, ACRONYM, ABBR

  • list: DD, DT

  • NOFRAMES, NOSCRIPT

  • ADDRESS, CENTER

    • Note: The style attribute for this interface is reserved for future usage.

    • contentEditable ( )
      returns DOM.DOMString
      KDE 4.1.0 and later only
      		Returns the kind of editability that applies to this element.

      The returned string is one of:

    • true: This element has been set to be editable.
    • false: This element has been set not to be editable.
    • inherit: This element inherits its editability from the parent.

      • This function is not part of the DOM specifications as defined by the w3c.
