class CSSPageRule
Module khtml
Namespace DOM
Class CSSPageRule
Inherits DOM::CSSRule
The CSSPageRule interface represents a page rule within a CSS style sheet. The @page rule is used to specify the dimensions, orientation, margins, etc. of a page box for paged media.
