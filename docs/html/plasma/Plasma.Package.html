Installs a package.

Parameters:
package  path to the Plasmagik package
packageRoot  path to the directory where the package should be installed to
servicePrefix  the prefix for the desktop file, so as not to interfere with unrelated services (eg: "plasma-applet-"). If no prefix is set (e.g. a QString() is passed in), then the package will NOT be registered as a service

Returns:
true on successful installation, false otherwise
