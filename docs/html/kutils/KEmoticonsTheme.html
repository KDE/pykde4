  • parseEmoticons (text, mode, exclude)
    returns QString
    ArgumentTypeDefault
    textQString
    modeKEmoticonsTheme.ParseModeKEmoticonsTheme.DefaultParse
    excludeQStringListQStringList()
    		Parse emoticons in text text with ParseMode mode and optionally excluding emoticons from exclude 
    KEmoticonsTheme theme = KEmoticons().theme();
QString text = ":D hi :)";
QStringList exclude(":)");

    text - the text to parse

    mode - how to parse the text

    exclude - a list of emoticons to exclude from the parsing Returns the text with emoticons replaced by html images SkipHTML is forced when using this function
