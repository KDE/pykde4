Represents a user on your system

This class represents a user on your system. You can either get information about the current user, of fetch information about a user on the system. Instances of this class will be explicitly shared, so copying objects is very cheap and you can safely pass objects by value.

  • KUser (mode)
    returns a KUser object
    ArgumentTypeDefault
    modeKUser.UIDModeUseEffectiveUID

    Creates an object that contains information about the current user. (as returned by getuid(2) or geteuid(2), taking $LOGNAME/$USER into account). #UseEffectiveUID is passed the effective user is returned. #UseRealUserID is passed the real user will be returned. The real UID will be different than the effective UID in setuid programs; in such a case use the effective UID for checking permissions, and the real UID for displaying information about the user.
