class Tag
Module nepomuk
Namespace Nepomuk
Class Tag
Inherits Nepomuk::Resource
\class Tag tag.h Nepomuk/Tag

A Tag can be assigned to any Thing. This allows simple grouping of resources. Each Tag is identifed by its label which should be unique.
