HTMLLinkElement Class Reference

from PyKDE4.khtml import *

Inherits: DOM.HTMLElement → DOM.Element → DOM.Node
Namespace: DOM

Detailed Description

The LINK element specifies a link to an external resource, and defines this document's relationship to that resource (or vice versa). See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#edef-LINK"> LINK element definition in HTML 4.0.

DOM.DOMString charset (   self )

The character encoding of the resource being linked to. See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#adef-charset"> charset attribute definition in HTML 4.0.

bool disabled (   self )

Enables/disables the link. This is currently only used for style sheet links, and may be used to activate or deactivate style sheets.

DOM.DOMString href (   self )

The URI of the linked resource. See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#adef-href"> href attribute definition in HTML 4.0.

DOM.DOMString hreflang (   self )

Language code of the linked resource. See the <a href="http://www.w3.org/TR/REC-html40/struct/links.html#adef-hreflang"> hreflang attribute definition in HTML 4.0.
