class ItemView
Module akonadi
Namespace Akonadi
Class ItemView
Inherits QTreeView
A view to show an item list provided by an ItemModel.

When a KXmlGuiWindow is set, the XMLGUI defined context menu akonadi_itemview_contextmenu is used if available.
