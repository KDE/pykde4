class KIconTheme
Module kdeui
Namespace
Class KIconTheme
Inherits
Internal Class to use/access icon themes in KDE. This class is used by the iconloader but can be used by others too. You should not use this class externally. This class is exported because the KCM needs it.

See also KIconLoader
