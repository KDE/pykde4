    KDE 4.1.0 and later only
    ArgumentTypeDefault
    urlsKUrl.List
    deletionTypeKIO::JobUiDelegate.DeletionType
    confirmationTypeKIO::JobUiDelegate.ConfirmationType
    		Ask for confirmation before deleting/trashing urls.

    Note that this method is not called automatically by KIO jobs. It's the application's responsibility to ask the user for confirmation before calling KIO.del() or KIO.trash().

    urls - the urls about to be deleted/trashed

    method - the type of deletion (Delete for real deletion, Trash otherwise)

    confirmation - see ConfirmationType. Normally set to DefaultConfirmation. Note: the window passed to setWindow is used as the parent for the message box. Returns true if confirmed
