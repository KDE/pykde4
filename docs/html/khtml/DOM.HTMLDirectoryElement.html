HTMLDirectoryElement Class Reference

from PyKDE4.khtml import *

Inherits: DOM.HTMLElement → DOM.Element → DOM.Node
Namespace: DOM

Detailed Description

Directory list. See the <a href="http://www.w3.org/TR/REC-html40/struct/lists.html#edef-DIR"> DIR element definition in HTML 4.0. This element is deprecated in HTML 4.0.
