ProcessingInstruction Class Reference

from PyKDE4.khtml import *

Inherits: DOM.Node
Namespace: DOM

Detailed Description

The ProcessingInstruction interface represents a &quot;processing instruction&quot;, used in XML as a way to keep processor-specific information in the text of the document.
