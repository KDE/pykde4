Getting Started

This tutorial traces the development of a simple KDE program.

Even experienced programmers may want to browse the source code for this tutorial, as KDE4/PyKDE4 have a number of changes from earlier versions, including the module which contains KApplication, the use of ki18n and KLocalizedString, the semantics of actions and enums, and the absence of the "plug" method for actions.

The sections in the tutorial are:

  • Hello, World
  • Main Window
  • KXmlGui Main Window
  • Action!
