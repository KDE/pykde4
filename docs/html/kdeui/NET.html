Base namespace class.

The NET API is an implementation of the NET Window Manager Specification.

This class is the base class for the NETRootInfo and NETWinInfo classes, which are used to retrieve and modify the properties of windows. To keep the namespace relatively clean, all enums are defined here.

See also http://www.freedesktop.org/standards/wm-spec/

  • timestampCompare (time1, time2)
    returns (int)
    static
    ArgumentTypeDefault
    time1ulong
    time2ulong

    Compares two X timestamps, taking into account wrapping and 64bit architectures. Return value is like with strcmp(), 0 for equal, -1 for time1 < time2, 1 for time1 > time2.

  • timestampDiff (time1_, time2_)
    returns (int)
    static
    ArgumentTypeDefault
    time1_ulong
    time2_ulong

    Returns a difference of two X timestamps, time2 - time1, where time2 must be later than time1, as returned by timestampCompare().

    • State = { Modal, Sticky, MaxVert, MaxHoriz, Max, Shaded, SkipTaskbar, KeepAbove, StaysOnTop, SkipPager, Hidden, FullScreen, KeepBelow, DemandsAttention }
      Modal - indicates that this is a modal dialog box. The WM_TRANSIENT_FOR hint MUST be set to indicate which window the dialog is a modal for, or set to the root window if the dialog is a modal for its window group. /p> p>
      Sticky - indicates that the Window Manager SHOULD keep the window's position fixed on the screen, even when the virtual desktop scrolls. Note that this is different from being kept on all desktops. /p> p>
      MaxVert - indicates that the window is vertically maximized. /p> p>
      MaxHoriz - indicates that the window is horizontally maximized. /p> p>
      Max - convenience value. Equal to MaxVert | MaxHoriz. /p> p>
      Shaded - indicates that the window is shaded (rolled-up). /p> p>
      SkipTaskbar - indicates that a window should not be included on a taskbar. /p> p>
      KeepAbove - indicates that a window should on top of most windows (but below fullscreen windows). /p> p>
      StaysOnTop - b style="color : #ff0000;">Deprecated This is an obsolete name for KeepAbove. /p> p>
      SkipPager - indicates that a window should not be included on a pager. /p> p>
      Hidden - indicates that a window should not be visible on the screen (e.g. when minimised). Only the window manager is allowed to change it. /p> p>
      FullScreen - indicates that a window should fill the entire screen and have no window decorations. /p> p>
      KeepBelow - indicates that a window should be below most windows (but above any desktop windows). /p> p>
      DemandsAttention - there was an attempt to activate this window, but the window manager prevented this. E.g. taskbar should mark such window specially to bring user's attention to this window. Only the window manager is allowed to change it. /p> p>

      Unknown - indicates that the window did not define a window type. /p> p>
