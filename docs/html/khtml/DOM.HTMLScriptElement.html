HTMLScriptElement Class Reference

from PyKDE4.khtml import *

Inherits: DOM.HTMLElement → DOM.Element → DOM.Node
Namespace: DOM

Detailed Description

Script statements. See the <a href="http://www.w3.org/TR/REC-html40/interact/scripts.html#edef-SCRIPT"> SCRIPT element definition in HTML 4.0.
